Technology Stack

Android

Android Studio is an integrated development environment (IDE) for developing applications for Google’s mobile operating system, Android. It includes tools for code editing, compiling, debugging, and publishing apps on Google Play.

Google Fit SDK

Google Fit SDK is an open source library that allows developers to integrate Google Fit features into their apps. It also includes APIs for accessing data from the device.

Java

Java is a general purpose computer programming language used for developing applications across multiple platforms. Java is one of the most popular languages used today because it allows programmers to write programs quickly and easily.

Xamarin Native C

Xamarin Native is a cross-platform mobile application development toolkit from Microsoft. It lets you build native apps for iOS, Android, and Windows 10 Mobile. Xamarin Native lets you build cross-platform mobile apps using C# and Visual studio. You can use the same code base to develop apps for multiple platforms such as iOS, Android, and Windows Phone.

React Native

React Native is an open source framework that allows developers to build cross platform applications using JavaScript. This means that you can use React Native to create native mobile apps for both iOS and Android. The framework also provides access to native APIs on each device.

Firebase

Firebase is a cloud-based platform that makes building mobile apps easy. We'll walk through the process of creating a realtime chat app using Firebase. Firebase is used by companies such as Instagram, Pinterest, and Uber. It's also used by startups and small businesses who need to quickly develop features into their products.

Unreal Engine

Unreal Engine 4 is a powerful game engine for creating 3D games. It has been used by many popular video games. Unreal Engine is a free cross platform game engine developed by Epic Games. It is available for Windows, Mac OS X, Linux, iOS, Android, Xbox One, PlayStation 4, Nintendo Switch, Wii U, Oculus Rift, and HTC Vive.

Unity Engine

A Unity Engine is a software development kit that allows developers to use the Unity game engine to create video games. The engine is free, open source and cross-platform and it runs in the Microsoft Windows, OS X and Linux operating systems.

ARToolkit

ARToolKit is a collection of high-quality, performance-intensive iOS Components. The ARToolkit components are designed to simplify your application development and increase application performance. ArtKit is an open-source framework for developing modern mobile applications using iOS and Mac OS X technologies. It's designed to work with the Apple technology stack, including iOS, macOS, watchOS, tvOS and even iMessage apps.

HTML5

With HTML 5, game developers have the ability to add new functionality to their games without having to rewrite code. This allows them to keep their games up-to-date with changes in technology and other areas of the industry.

Photon

Photon is a decentralized platform that connects you to the world of crypto currency and blockchain. The Photon platform is a decentralized blockchain network that enables users to trade and invest in cryptocurrencies, tokens and other digital assets. The platform allows users to create their own cryptocurrency exchange, trading desk or investment fund.

Python

Python is an object-oriented computer programming language. It has a large and loyal community of developers, and its syntax is highly readable and more flexible than many other languages.

Vuforia

A Vuforia is an SDK (software development kit) that allows developers to create augmented reality (AR) apps. The SDK allows developers to create AR experiences for iOS and Android devices, as well as Windows 10 PCs and devices with the HoloLens headset.
